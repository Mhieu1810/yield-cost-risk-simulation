# Yield & Cost Risk Simulation (Monte Carlo)

## Overview
This project simulates manufacturing yield-related costs under uncertainty
using Monte Carlo simulation.

It maps process capability (Cpk) to scrap rate,
evaluates cost variability, and identifies risk drivers.
